# ポータルWebサイト(フロントエンド用)

## summary

このWebアプリケーションは、私のプロジェクトやポートフォリオを一元的に紹介するポータルサイトです。<br>
開発における各プロジェクトを整理し、簡単にアクセスできるようにするために構築しました。

## Running

Webアプリケーションを起動するには以下のコマンドを実行してください。

```bash
npm run dev
```
